How it works

Simple input flow for desktop play.

Sky Player reads a song file, maps the notes to a PC playback flow, and helps automate the timing so you can focus on the music session.

1

Choose a song

Pick a supported JSON, skysheet, or compatible TXT file from your local song collection.

2

Preview the workflow

Use the desktop-friendly interface to prepare playback before sending notes to the game.

3

Start playback

Run the song with automatic note timing for a smoother PC music playing experience.

Responsible use Sky Player is a community tool for music playback workflows on PC. Automated playback or simulated input may not be allowed in every game context. Review the rules for Sky: Children of the Light and use the project responsibly at your own risk.
