Why Sky Player

Why not just any macro?

Playing a full song by hand is exhausting, and generic auto-clickers weren't built for music — they drift off-beat, mash chords into broken arpeggios, and cut long notes short. Sky Player is built specifically for Sky's instrument, so what you hear is the song, not a stumble.

Keeps the beat

Every note fires on time, from slow ballads to rapid-fire passages. Speed a song up or slow it down and it still lands where it should.

Chords, not clatter

Notes meant to sound together are struck together in a single game frame, so chords come out clean instead of a smeared roll of keys.

Your setup, your call

Per-song profiles let you tweak tempo, frame rate, and theme. There's even a dry-run mode to preview a song without sending a single keystroke.

Reliable playback

Steady, even when your PC isn't

Background downloads, antivirus scans, a loading spike in-game — the kind of things that make hand-played songs stumble. Sky Player is built to keep its timing through all of it.

Light on your machine

It sips about 100 MB of RAM and stays quiet in the background, so it won't fight your game for resources or cost you frames.

Holds the tempo

The timing engine is tuned to stay on beat even under load, so a busy PC doesn't turn your performance into a mess of early and late notes.

Tunes to your rig

Run --doctor to check your system, then pick a tuning preset. Even a modest laptop can play cleanly with the right profile.

Friends hearing dropped notes?

That's usually network packet loss, not Sky Player — the keys leave on time, some just never reach Sky's servers. The free Cloudflare WARP (1.1.1.1) client on "Traffic and DNS (UDP)" mode routes game traffic over a cleaner path and usually fixes it.

Safety & updates

Built to stay on the safe side

Sky Player works from the outside like any keyboard would, and it's open source — so you don't have to take our word for it. The security rules are even re-checked automatically before any change ships.

Never touches the game

No memory reads, no DLL injection, no debuggers, no edited game files. Sky Player runs entirely as its own separate program.

Just keystrokes

Input goes through Windows' built-in SendInput — the exact same path a normal keyboard or macro uses. Nothing exotic.

Bad files can't bite

Every sheet is validated before it plays. A broken or malformed file is caught early with a clear message instead of misfiring.

Updates you control

The app never rewrites itself while running — it just flags new versions. You close it and run updater.bat, which verifies the download's SHA256 before copying and rolls back on error. Your config.json and songs folder are left untouched.

Get started

Play your first sheet in minutes

Sky Player is portable — no installer, no admin rights, no registry settings. Just extract and run.

  1. Download

    Grab the latest zip from GitHub Releases and unzip it anywhere — Desktop, Documents, wherever.

  2. Add songs

    Export sheets from the Sky Music editor as .json, .skysheet, or .txt and drop them into the songs folder.

  3. Open the player

    Run Sky-Player.exe in Windows Terminal. Type to fuzzy-search your library; Ctrl+R reloads it after adding songs.

  4. Arm a song

    Highlight the track you want and press Enter. Tweak tempo, FPS, or theme first with t / f / y if you like.

  5. Play & control

    Switch to the Sky window and it plays itself. F8 pause · F9 skip · F10 stop · F6 re-focus the game.

Supported formats

Load your sheets, your way

Sky Player accepts the three most common Sky sheet formats — no conversion needed. Sheets made in the popular Sky Music web editor (specy/skyMusic) work out of the box.

JSON

Standard JSON

The de-facto community format. Supports notes, chords, holds, tempo changes, and metadata (title, author, BPM).

skysheet

Skysheet (.skysheet)

Community format from the Sky Music web editor ecosystem. Structurally JSON-based with metadata conventions. Full feature parity with standard JSON — notes, chords, holds, tempo changes.

TXT

JSON-compatible TXT

Plain-text files containing valid JSON. Useful for quick edits in Notepad or sharing sheets in chat without file attachments.

Frequently asked

Things people ask before downloading

Will I get banned for using Sky Player?

Sky Player stays completely outside the game — it only sends standard keystrokes through the public Windows SendInput API, just like a keyboard macro or accessibility tool. It never reads game memory, hooks processes, injects DLLs, or modifies game files, so there's nothing for an anti-cheat to catch. That said, be honest with yourself: automated music playback may still go against Sky's Terms of Service. Use it responsibly and at your own risk.

Where do I get song sheets?

The Sky Music web editor (specy/skyMusic) is the main source — pick a song, export it as JSON, skysheet, or txt, and drop the file into your songs folder. The Sky Player GitHub Discussions also has community-shared sheets ready to download.

Why do my friends hear dropped or stuttering notes?

Usually it's not Sky Player — the keypresses leave your PC on time, but network packet loss between you and Sky's servers means some never arrive. Installing the free Cloudflare WARP (1.1.1.1) client and setting it to "Traffic and DNS (UDP)" usually routes your game traffic over a cleaner path and fixes it.

Is Sky Player free?

Yes — 100% free and open source under the GNU GPL v3.0 license. No signups, no ads, no trackers. Download the latest build directly from GitHub Releases, verify the SHA256 sidecar if you like, and run it.

Which Windows versions are supported?

Sky Player runs on 64-bit Windows 10 and Windows 11. It is a portable application that runs directly from its folder without requiring system installers, admin rights, or registering background services.

What file formats can I load?

You can load JSON (.json), skysheet (.skysheet), and plain text (.txt) files containing JSON-compatible lists. All community sheets created in standard editors like Sky Music web editor work natively without conversions.

How do updates work?

The application checks for new releases and shows a simple in-app notice when a version is available. To apply an update, close the app and run the updater.bat utility. The external script verifies SHA256 checksums before applying any files and preserves your config.json and custom songs/ directories automatically.

Does Sky Player work on Mac or Linux?

No. Sky Player relies on Windows-only mechanisms — SendInput for keystrokes and MMCSS for precise timing — so macOS and Linux aren't supported and aren't on the roadmap.
