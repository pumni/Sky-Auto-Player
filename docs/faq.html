Sky Auto Player / FAQ

Frequently Asked Questions

Everything people ask about Sky Auto Player - the free, open-source Windows tool that plays Sky: Children of the Light music sheets hands-free through the public Windows SendInput API.

Is Sky Auto Player free?

Yes. Sky Auto Player is 100% free and open source under the GNU GPL v3.0 license. Download the latest packaged release from GitHub Releases - no signup, no ads, no telemetry.

Will I get banned for using Sky Auto Player with Sky: Children of the Light?

Sky Auto Player stays completely outside the game. It only sends standard keystrokes through the public Windows SendInput API - the same channel any keyboard macro uses - and never reads game memory, injects code, hooks processes, attaches debuggers, or modifies game files. Note that automated music playback may still go against Sky's Terms of Service, so use it responsibly and at your own risk. See SECURITY.md for the full mandate list.

What file formats does Sky Auto Player support?

Sky Auto Player plays JSON (.json), skysheet (.skysheet), and JSON-compatible TXT (.txt) sheets exported from the Sky Music web editor (specy/skyMusic). Drop them in the songs/ folder next to Sky-Auto-Player.exe and press Ctrl+R in the picker to reload the list.

Which Windows versions does Sky Auto Player support?

Sky Auto Player runs on Windows 10 and Windows 11 (64-bit). It is a portable application - no installer, no admin rights, no background services. Windows 7 and Windows 8 are not supported.

Where can I download Sky Auto Player?

Download Sky Auto Player from the GitHub Releases page. Every release ships with a SHA256 sidecar so you can verify the download before running it.

Does Sky Auto Player work on Mac or Linux?

No. Sky Auto Player relies on Windows-specific APIs - SendInput for input simulation and MMCSS for real-time thread scheduling. Mac and Linux are not on the roadmap.

What is the difference between JSON, skysheet, and TXT sheet files?

All three encodings describe the same set of notes, chords, and holds. JSON (.json) is the canonical structured form. Skysheet (.skysheet) is the Sky Music web editor's compact export format. TXT (.txt) files that contain a JSON payload are also accepted for users who keep sheets in plain-text workflows. Sky Auto Player auto-detects the format from the payload content, not from the file extension.

How does Sky Auto Player update itself?

Sky Auto Player never overwrites its own binaries while running. It only notifies you when a new release is available. Applying the update is one explicit step: close the app and run updater.bat in the install folder. The updater verifies the SHA256 of the downloaded zip against its sidecar before touching any file, stages the download in %TEMP%, copies binaries transactionally with rollback, and never replaces config.json or the songs/ folder. The only config keys it may patch are update.last_check_ts and update.last_notified_version.

Why are some notes dropped during playback?

Dropped notes are usually network packet loss between your PC and Sky's servers, not a Sky Auto Player timing bug - the keypresses leave your machine on time, but a few never reach the game server. Installing the free Cloudflare WARP (1.1.1.1) client and switching it to "Traffic and DNS (UDP)" mode routes game traffic over Cloudflare's backbone and usually fixes the problem.

Can I tune Sky Auto Player for a weak PC?

Yes. Run Sky Auto Player with the --doctor flag to check your GIL state, MMCSS availability, and key mapping. Then pick a preset from docs/tuning-presets.md - including the free-threaded python3.14t interpreter, balanced, and audience_safe profiles.

Can I build Sky Auto Player from source?

Yes. Clone the repository and run uv sync to set up the Python 3.14 free-threaded environment. See docs/tuning-presets.md for non-standard environment presets and Sky-Auto-Player.spec for the PyInstaller build configuration.

Does Sky Auto Player need admin rights?

No. Sky Auto Player is fully portable, runs from any folder, and writes no registry entries. Admin rights are never required for the app itself; only the external updater.bat may need write permission to its own install folder.

Where are Sky Auto Player's logs and config stored?

Config and songs live alongside Sky-Auto-Player.exe in the install folder (a portable config.json and a songs/ directory). The external updater writes one append-only line per run to %LOCALAPPDATA%\Sky-Auto-Player\updater.log. No personal information is logged.

Is Sky Auto Player related to the Sky Music web editor?

They are separate projects by separate authors, but they form a workflow pair. specy/skyMusic is the web editor used to author and export song sheets. Sky Auto Player is the desktop tool that plays those sheets inside Sky: Children of the Light on Windows.
